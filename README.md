Vendor Performance Analysis

This project presents a comprehensive, end-to-end data analytics workflow focused on evaluating vendor performance and optimizing profitability in a restaurant/retail business environment. The goal is to simulate a real-world business scenario where organizations rely on data-driven insights to improve vendor selection, pricing strategies, and inventory efficiency.

The analysis is built on multi-source structured data, including sales transactions, purchase records, and vendor invoices. These datasets are integrated using SQL and transformed into a unified analytical dataset through a structured ETL process. This approach reflects how data is handled in production environments, where information is distributed across multiple tables and requires consolidation for meaningful analysis.

A key aspect of this project is the creation of a centralized aggregated dataset, combining relevant features such as purchase quantities, sales performance, pricing details, and vendor-level information. The dataset is optimized to reduce complexity and enable efficient querying, ensuring scalability when working with large volumes of data.

Using Python (Pandas), advanced data analysis and feature engineering are performed to derive critical business metrics, including:

Gross Profit to measure overall profitability
Profit Margin (%) to evaluate efficiency of sales
Inventory Turnover to assess stock movement and operational efficiency

The analysis focuses on identifying:

High-performing vendors contributing significantly to revenue and profit
Underperforming vendors requiring pricing or promotional adjustments
Inefficiencies in pricing strategies affecting profitability
Inventory bottlenecks impacting operational performance

To communicate insights effectively, an interactive Power BI dashboard is designed, presenting key performance indicators (KPIs), trends, and comparisons across vendors and products. The dashboard enables stakeholders to quickly interpret data and make informed decisions regarding vendor partnerships, pricing optimization, and inventory management.

Overall, this project demonstrates the complete lifecycle of a data analytics solution—from raw data ingestion and transformation to insight generation and business reporting—closely aligning with real-world industry practices.

🔧 Tech Stack
SQL – Data extraction, joins, aggregations, and query optimization
Python (Pandas) – Data cleaning, transformation, and feature engineering
Power BI – Data visualization and dashboard development

🚀 Key Highlights
End-to-end ETL pipeline for handling large, multi-table datasets
Data modeling and aggregation for efficient analysis
Vendor-level profitability and performance evaluation
Business-driven insights for pricing and inventory optimization
Interactive dashboard for stakeholder reporting
